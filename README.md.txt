# Django Ecommerce Project

A simple ecommerce web application built using Django.

## Features

- User registration and login
- Product listing
- Product detail page
- Shopping cart
- Checkout system
- Order management
- Order history
- Admin panel for product management

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Project Structure

User → Orders → OrderItems → Products

## Installation

Clone the repository

git clone https://github.com/ytharun07-dev/django-ecommerce-project.git

Go to project folder

cd django-ecommerce-project

Install dependencies

pip install -r requirements.txt

Run migrations

python manage.py migrate

Start server

python manage.py runserver

## Admin Panel

http://127.0.0.1:8000/admin
